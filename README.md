## About

Weightfield2 is a freeware 2D simulator for silicon and diamond detectors

## Building
Clone the repo

    $ git clone https://github.com/TheRickyRoss2/Weightfield2.git
    
Make weightfield2 and run

    $ make -j8
    $ ./weightfield
